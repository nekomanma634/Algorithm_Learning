# Algorithm Learning

アルゴリズムの学習記録および、プログラミングコンテスト（AtCoder ABC/AHC等）の提出コードを管理するリポジトリです。

## 概要
* **目的**: アルゴリズムの理解、データ構造の実装、およびコンテストの過去問演習。
* **主な使用言語**: C++

## 開発環境
* **Compiler**: 
  - g++ (GCC) 12.2.0 or later

## 使い方
各問題のディレクトリへ移動し、コンパイルして実行します。

```bash
g++ -std=gnu++20 -O3 -o main main.cpp

./main
```
